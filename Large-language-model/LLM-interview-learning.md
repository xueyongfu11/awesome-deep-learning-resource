# LLM Interview Learning Resources

## Main Interview Notes

- https://github.com/wdndev/llm_interview_note
  - 中文大模型面试主线资料，覆盖 LLM 基础、训练、SFT、RLHF、RAG、Agent、显存、推理优化等。
  - 适合做第一轮系统复习，也适合按专题查漏补缺。
  - 项目实践包含 tiny-llm、tiny-rag、tiny-mcp 等，适合补充简历项目表达。

- https://github.com/km1994/LLMs_interview_notes
  - 中文 LLM 面试题库，覆盖 Decoder-only、Attention、LayerNorm/RMSNorm、SFT、RAG、PEFT、显存优化等。
  - 适合用于面试前刷题和自测。
  - 部分答案依赖外部文章，建议和论文、官方文档或其他资料交叉验证。

- https://github.com/315386775/DeepLearing-Interview-Awesome-2024
  - 综合型深度学习和大模型面试资料，覆盖深度学习基础、LLM、CV、项目、场景题等。
  - 适合算法岗、AIGC 算法岗、机器学习工程师岗位综合准备。

- https://github.com/EmbraceAGI/AIGC_Interview
  - AIGC 求职和面试资料集合，包含面经、Prompt、Embedding、Fine-tuning、Stable Diffusion、ChatGPT 等。
  - 适合快速了解岗位问题风格和面试高频方向。

## English Question Banks

- https://github.com/llmgenai/LLMInterviewQuestions
  - 英文 LLM 面试题集合，包含基础原理、训练、微调、推理、RAG、应用等方向。
  - 适合准备英文面试、外企面试，或用于补充中文资料没有覆盖的问题。

## Systematic Learning

- https://github.com/mlabonne/llm-course
  - 系统化 LLM 课程，分为 LLM Fundamentals、LLM Scientist、LLM Engineer。
  - 覆盖 Transformer、预训练、微调、量化、RAG、部署等。
  - 适合用来建立知识体系，不建议只当题库使用。

- https://github.com/WALLE-AI/uLMTutorial
  - 中文大模型入门到进阶教程，覆盖 Transformer、预训练、量化、RLHF、Prompt、RAG、Agent、多模态等。
  - 适合基础不牢时先补体系，再进入面试题训练。

## Project Practice For Interviews

- https://github.com/Snailclimb/interview-guide
  - 智能 AI 面试官项目，包含 Spring Boot、React、PostgreSQL、pgvector、RAG、语音等技术栈。
  - 适合 Java 后端 + AI 应用方向，用于准备 AI 应用项目经历。

- https://github.com/LeDat98/NexusRAG
  - RAG 实战项目，覆盖向量检索、知识图谱、重排序、文档解析、引用生成、多模态文档处理等。
  - 适合准备 RAG、知识库问答、企业级 AI 应用方向的项目面试。
